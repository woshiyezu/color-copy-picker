## Usage

1. Download or clone this repo.

2. Install dependencies.

```
$ npm install
```

3. Run project.
 
```
$ expo start
```

## Publishing

```
development:
$ npm run publish:dev

production:
$ npm run publish:prod
```

## Building Android App

```
development:
$ npm run android:dev

production:
$ npm run android:prod
```
